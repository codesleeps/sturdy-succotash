## ➤ Global Weather App 🌍

### Operation 🎯

 - Click the 'App' link below 
 - Enter the city you require in the input field to get the current weather data for that city  🌆
 
     - City & Date
     - Current Temperature
     - Expected High
     - Humidity
     - Wind Speed
     - UV Index
 
 
 [App](https://codesleeps.github.io/sturdy-succotash/) ☀️
