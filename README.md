# sturdy-succotash

## Global Weather App 🌍

### Operation 🎯

 - Simply enter the city you require in the input field to get current weather data for that city  🌆
 
     - City & Date
     - Current Temperature
     - Expected High
     - Humidity
     - Wind Speed
     - UV Index
 
 
 [App](https://codesleeps.github.io/sturdy-succotash/) ☀️
